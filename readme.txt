Tool too easily import and export all existing Spots as CSV files

== Description ==

This plugin adds an Import/Export option to the Spots menu that allows you to 
easily export Spots to a CSV file, and import them on another Spots enabled site

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
